# My-what-s-up-bot# My-what-s-up-bot 🤖

A lightweight WhatsApp bot built using **Node.js** and **Baileys**.

## 🚀 Features
* **Admin Tools:** `.kick`, `.add`, `.tagall`
* **Media:** Download music with `.song`
* **Connection:** Supports Pairing Code and QR scan
